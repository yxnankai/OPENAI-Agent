AI Agent 应用启动说明
====================

🎯 快速启动方法：

方法一：直接运行Python脚本
1. 确保已安装Python 3.8+
2. 安装依赖：pip install -r requirements.txt
3. 配置.env文件（设置OpenAI API密钥）
4. 运行：python main.py
5. 浏览器会自动打开 http://localhost:5000

方法二：构建exe文件（推荐）
1. 运行构建脚本：python build_simple.py
   或者双击：build_with_tray.bat
2. 等待构建完成
3. 双击运行 dist/AI_Agent.exe
4. 应用会在后台运行，系统托盘显示AI图标

🔧 主要功能：
- 语音交互（录音、播放、TTS）
- 网络搜索（多源搜索，返回详细结果）
- 本地工具（文件搜索、系统命令、网络查询）
- 文档读取（搜索和读取txt、doc、docx文件）
- 智能对话（基于OpenAI GPT模型）
- 模型切换（支持多种AI模型）

🎯 后台运行特性：
- exe文件运行时不会显示控制台窗口
- 应用在后台运行，系统托盘显示AI图标
- 右键点击托盘图标可以：
  * 打开浏览器访问应用
  * 查看运行日志
  * 退出应用
- 所有运行日志保存在 logs/ 目录中

⚠️ 注意事项：
1. 首次使用需要配置OpenAI API密钥
2. 确保麦克风和扬声器正常工作
3. 网络搜索需要网络连接
4. 系统命令有安全检查，危险操作需要确认
5. 如果看不到系统托盘图标，请检查任务栏设置
